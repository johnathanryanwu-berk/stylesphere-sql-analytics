# StyleSphere: E-commerce SQL Analytics Project

## Overview
This project simulates a full data pipeline for an e-commerce fashion retailer (StyleSphere). I designed and implemented a relational database, then used SQL and Python to analyze business performance.

## Key Features
- Designed normalized relational schema (3NF)
- Built database in PostgreSQL (Google Cloud SQL)
- Created SQL queries for business insights
- Integrated SQL with Python (psycopg2, pandas, plotly)
- Generated visualizations for decision-making

## Business Questions Answered
- Monthly revenue trends
- Top-performing product categories
- Payment success rates

## Tech Stack
- SQL (PostgreSQL)
- Python (pandas, psycopg2, plotly)
- Google Cloud SQL

## Files
- `StyleSphere_Project.pdf`: Full project documentation including schema, queries, and results

## Note
The database was deployed using Google Cloud SQL (free tier). While the live database is no longer active, all schema design, queries, and analysis are fully documented in the project.
